# 🚀 Angular Learning Journey

Raccolta di esercizi ed esperimenti che sto realizzando durante il mio percorso di apprendimento di **Angular** e **TypeScript** per la parte di Front-end, e **.NET** con **Entity Framework Core** per la parte di Back-end.

L'obiettivo è padroneggiare il framework partendo dalle basi fino a concetti più avanzati.

## 📂 Struttura del Repository

```text
angular-course-exercise/
├── Example-Api/                          # Esempio di Web API sviluppata in .NET che funge da backend
├── angular-demo/                         # Esercizio sulla creazione di components utilizzando Signal Inputs e Content Projection
├── angular-demo-API-httpResource/        # Esercizio con l'utilizzo di httpResource API
├── angular-demo-todolist/                # Esercizio sulla creazione di todo list con funzionalità per aggiungere, attivare e rimuovere
├── angular-multi-view/                   # Esercizio sulla gestione delle rotte e multi-view
└── angular-signal-forms/                 # Esercizio con l'utilizzo di Signal Forms
```
